# Welcome Geeks
<p>This is a 3D openGL based project.

<b>Langauge:</b> C++,OpenGL 
</b>Features:</b>
<p>
Camera Movement:
PRESS: w -MOVE forward
PRESS: a -MOVE left
PRESS: s -MOVE backward
PRESS: d -MOVE right
PRESS: r - auto rotate
PRESS: esc - auto rotate off
PRESS: q -MOVE down
PRESS: e -MOVE up
PRESS: k - rotate left
PRESS: l - rotate right
PRESS: i - zoom in
PRESS: o - zoom out
</p>
<p>
<b>Lighting Features:</b>
<ul>
    <li>GL_AMBIENT</li>
    <li>GL_DIFFUSE</li>
    <li>GL_SPECULAR</li>
    <li>GL_SHININESS</li>
</ul>
</p>
